# Email OTP Verification System

## 📌 Overview
This project is a web-based application that implements secure user registration using Email OTP verification.

## 🚀 Features
- User registration with email OTP verification
- OTP validation with error handling
- Secure data storage using Hibernate
- JSP-based user interface

## 🛠️ Tech Stack
- Java (Servlets, JSP)
- Hibernate
- MySQL
- HTML, CSS

## 🎯 Purpose
To demonstrate secure authentication and backend development skills.
